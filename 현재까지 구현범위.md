AI 기반으로 도로 위 낙하물을 탐지하고,
사용자에게 위험 정보를 제공하는 웹 서비스.

사용자 → (Frontend) → API → Service → Repository → DB
                              ↓
                           AI 모델

👤 사용자 기능 (User)

1. 회원 기능
회원가입
로그인 / 로그아웃
회원정보 수정
회원 탈퇴

이미지 / 영상 업로드
낙하물 탐지 (AI 분석 자동 실행)

# 관리자 기능 설명

관리자 페이지는 사용자 신고 데이터와 AI 탐지 결과를 효율적으로 관리하고,  
실시간 위험 상황에 빠르게 대응할 수 있도록 구성되어 있다.

## 1. 대시보드
관리자 페이지의 메인 화면이다.  
전체 신고 현황, 처리 상태, 위험 등급 통계 등 주요 정보를 한눈에 확인할 수 있다.  
시스템 전반의 운영 상태를 빠르게 파악하는 용도로 사용된다.

## 2. 회원 관리
가입한 사용자 목록과 계정 정보를 조회·관리하는 기능이다.  
관리자는 사용자 정보를 확인하고, 필요 시 권한 상태를 점검할 수 있다.

## 3. 권한 신청 관리
일반 사용자의 관리자 권한 신청 내역을 확인하고 승인 또는 반려할 수 있는 기능이다.  
권한 부여 과정을 관리하여 관리자 접근을 통제한다.

## 4. 신고 관리
사용자가 등록한 낙하물 신고 내역을 조회하고 관리하는 기능이다.  
신고 제목, 위치, 상태, 위험도, 첨부 파일, AI 탐지 결과 등을 확인할 수 있으며  
필요 시 처리 상태를 변경할 수 있다.

## 5. 실시간 위험 알림
AI 탐지 결과 중 위험/긴급 등급으로 분류된 상황을 실시간으로 확인하는 기능이다.  
Socket 기반으로 알림이 즉시 전달되며, 관리자는 이를 통해 빠르게 위험 상황에 대응할 수 있다.  
알림은 미확인/읽음 완료 상태로 구분하여 관리할 수 있다.

## 6. 위험 지역 분석
신고 위치와 탐지 데이터를 기반으로 위험 지역을 분석하는 기능이다.  
지도 기반 시각화를 통해 특정 지역의 위험도와 신고 분포를 확인할 수 있으며,  
지역별 위험 패턴을 파악하는 데 활용된다.

## 7. AI 분석 대시보드
AI 탐지 결과를 종합적으로 확인하는 기능이다.  
탐지 객체, 신뢰도, 분석 결과 요약 등을 통해 모델 성능과 탐지 현황을 확인할 수 있다.  
AI 기반 탐지 시스템의 운영 결과를 관리자 관점에서 검토하는 용도로 사용된다.
